# 📱 Mobile Sales Analytics Dashboard

## 📌 Business Problem

Businesses often struggle to gain actionable insights from large sales datasets due to fragmented reporting and limited visibility into key performance metrics. This project addresses these challenges by transforming raw sales data into interactive Power BI dashboards, enabling users to monitor KPIs, analyze sales trends, compare year-over-year (YoY) performance, and make informed business decisions.

---

## 🚀 Project Overview

This project demonstrates the complete Power BI development workflow, from data preparation and modeling to DAX calculations and interactive dashboard creation. The solution provides comprehensive insights into sales performance, customer behavior, product trends, regional sales, and payment preferences.

---

## 🔑 Key Contributions

- Cleaned and transformed raw sales data using **Power Query** by removing redundant columns, correcting data types, and improving data quality.
- Built a relational data model by creating relationships between fact and dimension tables, including a custom **Date (Calendar) Table** for time-intelligence analysis.
- Developed reusable **DAX measures** and calculated columns for key KPIs, including Total Sales, Total Transactions, Average Selling Price, Total Quantity Sold, and Month-to-Date (MTD) Sales.
- Implemented interactive **slicers, filters, and drill-through functionality** for dynamic analysis across brands, mobile models, payment methods, and time periods.
- Designed and developed **three interactive Power BI dashboards** to visualize KPIs, sales trends, regional performance, customer ratings, payment methods, product performance, and Year-over-Year (YoY) sales comparisons.
